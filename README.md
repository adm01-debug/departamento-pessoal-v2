# Sistema de Departamento Pessoal

Sistema completo para gestão de departamento pessoal.

## Funcionalidades

- Gestão de Colaboradores
- Folha de Pagamento
- Férias
- Ponto Eletrônico
- Benefícios
- eSocial
- Relatórios

## Tecnologias

- React 18
- TypeScript
- Vite
- TailwindCSS
- shadcn/ui
- React Query
- React Hook Form + Zod
- Vitest

## Instalação

```bash
npm install
npm run dev
```

## Testes

```bash
npm test
npm run test:coverage
```

## Build

```bash
npm run build
```
