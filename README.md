# Sistema de Departamento Pessoal

## Stack
- React 18 + TypeScript + Vite
- TailwindCSS + shadcn/ui
- React Query + Zustand
- React Hook Form + Zod

## Features
- Gestão de Colaboradores
- Folha de Pagamento
- Férias
- Ponto Eletrônico
- Benefícios
- eSocial
- Relatórios

## Quick Start

```bash
npm install
npm run dev
```
