- Button
- Card
- Dialog
- DataTable
- Form
- Input
- Select
- Badge
- Toast

## Common Components

### PageHeader
Cabeçalho padrão das páginas.

```tsx
<PageHeader
  title="Colaboradores"
  description="Lista de colaboradores"
  icon={Users}
  actions={<Button>Novo</Button>}
/>
```

### LoadingSpinner
Indicador de carregamento.

### EmptyState
Estado vazio com ícone e mensagem.

### SearchInput
Campo de busca com debounce.
