# Cálculo de Rescisão Trabalhista

## Tipos de Rescisão

### 1. Sem Justa Causa
- Aviso prévio (trabalhado ou indenizado)
- Saldo de salário
- 13º proporcional
- Férias vencidas + 1/3
- Férias proporcionais + 1/3
- Multa 40% FGTS

### 2. Por Justa Causa
- Saldo de salário
- Férias vencidas + 1/3

### 3. Pedido de Demissão
- Saldo de salário
- 13º proporcional
- Férias proporcionais + 1/3

## Fórmulas

### Aviso Prévio
```
Dias = 30 + (3 × Anos de Serviço)
Máximo = 90 dias
```

### 13º Proporcional
```
13º = (Salário / 12) × Meses Trabalhados
```

### Férias Proporcionais
```
Férias = (Salário / 12) × Meses × 1.33
```

## Documentos Gerados
- TRCT
- Guia FGTS
- Seguro Desemprego (se aplicável)
