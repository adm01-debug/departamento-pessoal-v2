# 📊 MONITORING.md

## Stack de Monitoramento

### Aplicação
- Sentry: erros e exceções
- LogRocket: session replay
- Hotjar: heatmaps

### Infraestrutura
- Datadog: métricas servidor
- PagerDuty: alertas
- Uptime Robot: disponibilidade

### Métricas Chave
- Error rate < 0.1%
- Uptime > 99.9%
- Response time < 200ms
- Apdex > 0.95
