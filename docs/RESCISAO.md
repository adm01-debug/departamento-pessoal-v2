# Rescisão

## Tipos

### Sem Justa Causa
- Aviso prévio
- Saldo de salário
- Férias proporcionais + 1/3
- Férias vencidas + 1/3
- 13º proporcional
- Multa 40% FGTS
- Saque FGTS
- Seguro desemprego

### Justa Causa
- Saldo de salário
- Férias vencidas + 1/3

### Pedido de Demissão
- Saldo de salário
- Férias proporcionais + 1/3
- Férias vencidas + 1/3
- 13º proporcional

### Acordo (Art. 484-A)
- 50% do aviso prévio
- Saldo de salário
- Férias proporcionais + 1/3
- Férias vencidas + 1/3
- 13º proporcional
- Multa 20% FGTS
- Saque 80% FGTS
