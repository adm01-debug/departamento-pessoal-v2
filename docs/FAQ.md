# FAQ - Perguntas Frequentes

## Geral

### Como faço login?
Acesse a URL do sistema e use seu email e senha cadastrados.

### Esqueci minha senha
Clique em "Esqueci minha senha" na tela de login.

## Colaboradores

### Como cadastrar um novo colaborador?
Menu > Colaboradores > Novo Colaborador

### Como gerar a folha de pagamento?
Menu > Folha > Processar Folha

## eSocial

### Quais eventos são suportados?
S-1000 a S-2299, S-2300 a S-2399, S-3000, S-5000 a S-5013

### Como enviar eventos?
Menu > eSocial > Transmitir

## Suporte

### Como reportar um bug?
Abra uma issue no GitHub com o template de bug.

### Como solicitar uma feature?
Abra uma issue com o template de feature request.
