# ADR 004: Estratégia de Testes

## Status
Aceito

## Contexto
Precisamos garantir qualidade com testes automatizados.

## Decisão
- **Unit:** Vitest + React Testing Library
- **Integration:** Vitest com MSW
- **E2E:** Playwright
- **Coverage:** Mínimo 80%

## Consequências
- Alta cobertura de testes
- Testes rápidos com Vitest
- E2E confiáveis com Playwright
