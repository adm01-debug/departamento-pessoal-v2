# ♿ ACCESSIBILITY.md - Guia de Acessibilidade

## Conformidade WCAG 2.1 AA

### Princípios
1. **Perceptível** - Alternativas textuais, contraste
2. **Operável** - Teclado, tempo, navegação
3. **Compreensível** - Legibilidade, previsibilidade
4. **Robusto** - Compatibilidade

### Implementações
- Skip links
- ARIA labels
- Focus management
- Alto contraste
- Leitor de tela compatível

### Testes
- axe-core automatizado
- Testes manuais com NVDA/VoiceOver
- Validação de contraste
